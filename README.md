# 考研笔记

这是一个使用Hugo构建的考研笔记网站，包含以下内容：

- 微积分笔记
- 线性代数笔记
- 概率论与数理统计笔记
- 控制理论笔记
- 控制理论习题
- 英语一大作文讲解
- 英语一小作文讲解

## 项目结构

- `content/`：存放Markdown格式的笔记内容
- `static/`：存放静态资源，如图片等
- `themes/`：存放Hugo主题
- `hugo.toml`：Hugo配置文件

## 本地开发

1. 安装Hugo
2. 克隆仓库
3. 运行 `hugo server -D` 启动本地服务器
4. 访问 `http://localhost:1313/` 查看网站

## 部署

项目使用GitHub Actions自动部署到GitHub Pages。
